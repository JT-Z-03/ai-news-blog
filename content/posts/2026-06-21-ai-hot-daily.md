## 今日头条

### 微软双向转售 GPT 与 DeepSeek，AI 模型分发开始更像“云中介”竞争

AI HOT 汇总的当日消息显示，微软正在同时扮演两端模型渠道商：一边通过 Azure 向中国客户提供 GPT 系列能力，另一边测试并准备向西方客户提供 DeepSeek-R1 与 DeepSeek-V4。这说明大模型竞争不再只是谁训练出更强模型，也是谁能把不同模型以合规、可商用的方式卖进更多市场。对企业客户而言，模型供应链正在从“单一厂商绑定”转向“多模型中间层采购”。

## 技巧与观点

### 《Deep Agents 实战》开源教程发布，重点落在真实 Agent 的上下文工程

这套教程由 LangChain 官方社区大使推出，围绕 LangChain、LangGraph 与 Deep Agents Harness，系统讲如何搭建可落地的 Agent 应用。它把重点放在 Runtime、Framework、Harness 三层结构，以及虚拟文件系统、任务规划、子 Agent 委派、Skills 复用等核心能力上。比起“再讲一遍 Agent 概念”，这更像一份面向真实工程实践的课程地图，适合已经开始做 Agent 产品、但需要把工作流做深做稳的团队。

🔗 [GitHub：datawhalechina/deepagents-in-action](https://github.com/datawhalechina/deepagents-in-action)
