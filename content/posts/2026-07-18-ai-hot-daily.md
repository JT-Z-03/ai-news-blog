## 产品发布/更新

### Claude Code v2.1.214 发布：修复权限绕过并改进多项权限检查

Claude Code v2.1.214 发布，重点修复 Windows PowerShell 5.1 中的权限检查绕过漏洞，并改进 Bash 权限检查在超长命令和 zsh 变量下标比较场景中的判断。此次更新还添加 EndConversation 工具，继续把编码 Agent 的安全边界、终端权限和会话控制放到更细的工程约束里处理。

### Index Ventures 联合创始人 Neil Rimer 认为 AI 财富将面临“再分配”

Index Ventures 联合创始人 Neil Rimer 表示，围绕 AI 积累的巨额财富将面临某种形式的再分配，无论这种再分配是自愿还是强制。他呼吁科技领袖主动推动自愿再分配，同时也提到美国慈善捐赠总额创新高但捐赠人数下降、加州考虑对亿万富翁征收一次性财富税等背景。AI 财富分配问题正在从抽象伦理讨论进入税收、慈善和公共政策的现实议程。
